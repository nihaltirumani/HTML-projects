Tables

Tables are used for organizing data in grid style. We should use <table> tag for starting a table. We can keep a caption to a table by using <caption> tag. To start the table's rows, we need to use <tr> tag. For the heading, we'll use <th> tag. For the content (data) we'll use <td> tag. You can see the below table constructed with the same sequence.
